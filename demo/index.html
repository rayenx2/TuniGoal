// ELT Pipeline — How It Works
1. Extract
extract_api.py fetches fixtures, standings, top scorers from api-sports.io v3 (league=202, season=2024). Falls back to 15 sample fixtures when no API key.
2. Stage
load_postgres.py bulk-inserts raw JSON into raw_matches, raw_standings, raw_topscorers with ON CONFLICT DO NOTHING idempotency.
3. Transform
transform_raw_spark.py runs PySpark on raw_matches: deduplication, null checks, negative-goals rejection, then builds dim_teams, dim_dates, dim_leagues, fact_matches via JDBC.
4. Delta load
Left-Anti join on match_id ensures zero duplicates across @daily reruns — only truly new records append to fact_matches.
5. Serve
FastAPI reads from both raw and gold tables. React shows standings, scorers, recent results, pipeline status, and the match outcome predictor.
